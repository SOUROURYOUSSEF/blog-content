After reading a shared posting from OMG!Ubuntu! about 'Share your Banshee Library with contacts on Empathy' I wanted to try this out on Fedora 12. The posting is about an extension for Banshee which allows contacts to share their playlist and music for download or streaming.

The build process on Fedora 12 is very simple. As root you need to run the following packages:

$ yum install git banshee mono-devel gtk-sharp2-devel banshee-devel


As user you can then compile the Banshee extension

$ git clone git://github.com/nloko/banshee-telepathy-extension.git
$ cd banshee-telepathy-extension
$ ./autogen.sh --enable-user-extension
$ make
$ make install


After you start Banshee it will show a Contacts group at the bottom of your Sources overview.
